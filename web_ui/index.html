TradingAgents

Multi-Agents LLM Financial Trading Framework.

On Olares, open the app from the launcher to use the web terminal (the CLI starts automatically). This page is only for health checks.

This app is CLI-based. To run analysis manually, use:

python -m cli.main

Set API keys via environment or .env (OPENAI_API_KEY, ANTHROPIC_API_KEY, etc.).
